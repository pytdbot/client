# Examples
Here is some examples of using [Pytdbot](https://github.com/pytdbot/client)

## userbot.py
> [!WARNING]
> **Using userbots is risky** and may result in your account being restricted or banned by Telegram. Only use a userbot on accounts that you can afford to lose. It is not recommended to use userbots for automation that violates Telegram's [Terms of Service](https://telegram.org/tos)

An example of logging in using a regular Telegram account (**userbot**) with Pytdbot.

## chatIDBot.py
An example of a bot that prints the ID of any chat selected by the user using `request_user/chat` button.

Real example: https://t.me/WhatChatIDBot.

## keyboardBot.py
An example of a bot that sends reply markup buttons.

## echobot.py
An example of a bot that echo content sent by the user.
